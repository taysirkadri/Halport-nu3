# KADRI Taysir — Developer Portfolio

A polished, interactive developer portfolio built with React, TypeScript, GSAP and Three.js.

## Overview

This repository contains the source for the personal portfolio site of **KADRI Taysir**. It demonstrates modern frontend techniques including 3D rendering with Three.js, motion with GSAP, and a responsive UI built with React + TypeScript.

## Demo / Preview

- Local development: `npm run dev`
- Production build: `npm run build`
- Preview build locally: `npm run preview`

Replace the preview image in `public/` or `src/assets/` to update the repository preview shown on GitHub.

## Features

- Interactive 3D character and scenes (Three.js)
- Smooth animations and scroll-driven effects (GSAP)
- Responsive layout and accessible navigation
- Modular React + TypeScript codebase

## Tech Stack

- React
- TypeScript
- GSAP
- Three.js
- Vite
- HTML & CSS

## Quick Start

Prerequisites: Node.js (16+) and npm or yarn.

1. Install dependencies

```bash
npm install
# or
yarn
```

2. Start the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

## Notes about GSAP Club Plugins

This project may reference GSAP Club plugins. Club plugins require a valid license for production hosting. If you see trial plugin code or placeholders, replace them with licensed plugin files or remove the plugin usage before deploying.
